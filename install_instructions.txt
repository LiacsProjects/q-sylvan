Build instructions for Ubuntu. (Should also works on Windows with WSL.)

(1) Clone/download repo

(2) Install dependencies (popt, GMP, MPFR, MPFR C++)
    - sudo apt-get install libpopt-dev
    - sudo apt-get install libgmp3-dev
    - sudo apt-get install libmpfr-dev
    - sudo apt-get install mpfrc++
    
(3) Build: In 'sylvan-qdd' directory:
    (3a) mkdir build
    (3b) cd build
    (3c) cmake ..
    (3d) make

(4) Some tests can be run from here with
    ./test/qdd_unit_tests
    ./quantum_algs/test
